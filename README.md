# Programação Funcional 

Repositório dedicado ao estudo e prática dos paradigmas da Programação Funcional. Aqui você encontrará:

### O que este repositório contém:
- **Exercícios resolvidos** em Haskell
- **Exemplos didáticos** dos principais conceitos:
  - Funções puras
  - Imutabilidade
  - Recursão
  - Higher-order functions
  - Pattern matching
  - Lazy evaluation
- **Comparativos** entre abordagens funcionais e imperativas
- **Projetos práticos** aplicando os conceitos

### Objetivos:
- Aprender os fundamentos do paradigma funcional
- Desenvolver o pensamento declarativo
- Entender vantagens como thread safety e código mais previsível
- Preparar para frameworks modernos (React, Elm, etc.)

### 💡 Por que Programação Funcional?
"Functional programming is like describing what you want, not how to get it" - [Anonymous]
